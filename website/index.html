FAST Engine by Eyevinn

The Sustainable FAST Channel Engine

Create unlimited number of FAST channels to a fraction of energy consumption compared to live transcoded FAST channels.

Running 100 FAST channels from a tiny instance? No problem. Try it out yourself:

docker run -d -p 8000:8000 -e DEMO_NUM_CHANNELS=100 eyevinntechnology/fast-engine

Based on VOD2Live technology and the open source Channel Engine library

  • Mix VOD and LIVE sources.
  • Passthrough of ad opportunity markers and proven integration with Server Side Ad Insertion providers.
  • Proven integration with origin repackagers to enable support for both HLS and MPEG-DASH.
  • Community provided plugins for integration with scheduling services.

By Eyevinn Technology AB and it is open source
